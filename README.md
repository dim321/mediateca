# README

Мультитенантная SaaS-платформа для управления рекламными аудио/видео материалами и их трансляции на устройства (ТВ) в торговых точках. Пользователи загружают медиафайлы, создают плейлисты, выкупают временные слоты устройств через аукцион. Администраторы управляют устройствами и группами. Реализация на Ruby on Rails 8 с Hotwire (Turbo + Stimulus), PostgreSQL, Active Storage + S3-совместимое хранилище, Solid Queue для фоновых задач.
